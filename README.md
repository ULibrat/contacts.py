Контактная книга

Это простая программа на Python, в которой можно сохранять контакты.

Что умеет программа:
добавлять новый контакт
редактировать контакт
удалять контакт
искать контакт по имени или номеру
показывать список всех контактов
Как это работает:

Программа сделана с помощью Tkinter (это для интерфейса) и SQLite (это база данных).
Все контакты сохраняются в файл contacts.db.

Как запустить:
Установить Python
Открыть файл с программой
Запустить его
Особенности:
обязательно нужно ввести имя и телефон
остальные поля можно оставить пустыми
данные сохраняются автоматически
